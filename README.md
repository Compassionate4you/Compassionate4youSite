# Compassionate4youSite

Team project for a home health and hospice website built with the PERN stack.

## Planned Features
- Landing page
- Home Health Services page
- Hospice Services page
- Patient/Staff Portal
- Manageable content and locations
- Accessibility options
  - Text size controls
  - Language options
  - High contrast mode
  - Screen reader support
- AI chatbot navigation/help

## Tech Stack
- PostgreSQL
- Express.js
- React
- Node.js
